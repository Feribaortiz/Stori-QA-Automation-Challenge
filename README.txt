# Stori-QA-Automation-Challenge
Stori-QA-Automation-Challenge- Fernando Ibarria

Use Robot Framework con selenium Library
Para el ejecutar el script se necesitan instalar lo siguiente
-Pip install robotframework
-pip install --upgrade robotframework-seleniumlibrary


Para ejecutar se debe abrir la consola en la siguiente carpeta
..\Stori-QA-Automation-Challenge\Stori\testcases
Y ejecutar el siguiente comando

robot prueba.robot

Con estos pasos se ejecutaran todos los scripts

El log se vera en 
..\Stori-QA-Automation-Challenge\Stori\testcases\log.html
